# UmaLab Data Directory

Place licensed historical race data here before production training.

Expected normalized CSV:

- `race_id`, `race_date`, `number`, `gate`
- `finish_position` or `is_win` and `is_place`
- feature columns from `app/feature_catalog.py`

Smoke data generated by `scripts/generate_synthetic_history.py` is only for verifying the pipeline. It is not real racing history and must not be used as a performance claim.

For the current local central-racing archive, use:

```bash
uv run python scripts/convert_keiba_data.py --input-dir data/keiba_data --output data/keiba_history_normalized.csv
```

The generated `*.csv` files are ignored by git.
